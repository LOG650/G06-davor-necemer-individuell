# References

## Struktur

- `01 forecasting/`
  Kilder om etterspørselsprognoser, tidsserier, sesongmodeller og SARIMAX.
- `02 capacity_planning/`
  Kilder om kapasitetsplanlegging, aggregate production planning, linear programming og relaterte modeller.
- `03 method/`
  Kilder om metode, forskningsdesign, validering og casebasert analyse.
