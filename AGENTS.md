# Repository Working Notes

## Folder Scope

Do not read or scan `000 templates/` during routine project startup or status checks unless the user explicitly asks for templates or template comparison.

Use the other project folders as the default source of project context:
- `001 info/`
- `002 meetings/`
- `003 references/`
- `004 data/`
- `005 report/`
- `011 fase 1 - proposal/`
- `012 fase 2 - plan/`
- `013 fase 3 - review/`
- `014 fase 4 - report/`

Rationale: `000 templates/` contains generic template material. The active project information is in the numbered project folders listed above.
